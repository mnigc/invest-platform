## 定义

**国内生产总值（GDP）** 是衡量一个国家或地区经济产出的最核心指标。

### 三种核算方法

- **生产法（Production Approach）**：GDP = 总产值 - 中间投入
- **支出法（Expenditure Approach）**：GDP = C + I + G + (X - M)
- **收入法（Income Approach）**：GDP = 工资 + 利润 + 租金 + 利息 + 折旧 + 间接税

### 名义 vs 实际

- **名义 GDP**：按当期价格计算
- **实际 GDP**：按基期价格计算，剔除通胀影响

### 构成分析

| 组成部分 | 含义 | 占比（美国） |
|---------|------|------------|
| C (消费) | 居民消费支出 | ~68% |
| I (投资) | 固定资本形成 | ~18% |
| G (政府) | 政府消费与投资 | ~17% |
| X-M (净出口) | 出口减进口 | ~-3% |

## 解读方法

1. **环比年化（Annualized QoQ）**：美国最常用，反映季度间的增长动能
2. **同比（YoY）**：中国最常用，反映年度增长
3. **GDP 平减指数**：名义 GDP / 实际 GDP，衡量整体价格水平变化

## 常见陷阱

- GDP 数据会多次修正（初值 → 修正值 → 终值），差异可达 1-2 个百分点
- 不同国家统计口径有差异（中国与美国的核算方法不同）
- 人均 GDP 比 GDP 总量更能反映经济福利
